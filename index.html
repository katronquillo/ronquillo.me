My name is Maria Katrina Ronquillo, but you can call me Katrina. I'm currently a second-year student at Carleton University studying Computer Science (Software Engineering, Co-Op) with a minor in Statistics.

Athens

Socket.io Javascript HTML/CSS Node.js

Athens is a simple trivia web application that makes use of the Open Trivia Database.

With Athens, users can create their own private games of trivia or join their friends using an existing game room's unique ID. Users also have the option of using the private or group chat functionality once they enter a game room.

ronquillo.me

HTML/CSS Javascript Bulma CSS

ronquillo.me is the web page that you are currently on! Using client-side Javascript, HTML/CSS, and the Bulma CSS Framework, I have developed this website with the intention of showcasing my existing skills and projects, while also giving myself the opportunity to learn along the way!

Sum Squares with MiniMax

Python Artificial Intelligence Abstract Data Types

A zero-sum, turn based game of Sum Squares that gives users the option to play against a MiniMax AI. This implementation of MiniMax was developed using the Tree and Queue datatypes.
